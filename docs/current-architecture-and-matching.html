4. AIマッチング処理フロー（詳細実装）

📥 STEP 1: リクエスト受信
API: POST /api/matching/recommend
入力: seeker_profile (スキル、経験、希望条件), available_jobs (求人リスト), top_k (返却数)
実装: app/api/endpoints/matching.py:27
🔍 STEP 2: 必須条件フィルタリング
処理: filter_by_requirements() - app/ml/matching_service.py:59
除外条件:
• ステータスが非公開 (status != "published")
• 除外職種カテゴリに該当
• 除外スキルタグが含まれる
必須条件:
• 勤務地が希望地域に含まれる
• 給与が希望最低年収以上
• 雇用形態が希望タイプに合致
結果: フィルタリング後の求人リスト
🧬 STEP 3: ベクトル埋め込み生成
モデル: Sentence Transformers (paraphrase-multilingual-MiniLM-L12-v2)
実装: app/ml/embedding_service.py:28
求職者プロフィール → テキスト:
"スキル: React, TypeScript | 経験: 3-5年 | 希望勤務地: 東京都"
求人 → テキスト:
"職種: Reactエンジニア | 仕事内容: ... | 勤務地: 東京都 | 必要スキル: React, TypeScript"
出力: 384次元のベクトル (np.ndarray)
📐 STEP 4: コサイン類似度計算
実装: app/ml/matching_service.py:41
計算式:
similarity = cosine_similarity(seeker_vector, job_vector)
範囲: 0.0 ~ 1.0 (完全不一致 ~ 完全一致)
ベーススコア: similarity × 80 (0-80点)
⭐ STEP 5: スキルマッチボーナス計算
実装: app/ml/matching_service.py:193
ロジック:
• 1スキル一致: +5点
• 2スキル一致: +10点
• 3スキル以上: +15~20点 (マッチ率に応じて)
上限: 20点
🎯 STEP 6: 最終スコア算出
計算式:
match_score = base_score (0-80) + skill_bonus (0-20)
match_score = min(match_score, 100)
例:
• ベクトル類似度: 0.75 → 60点
• スキル3つ一致 → +18点
• 最終スコア: 78点
💬 STEP 7: マッチング理由生成
実装: app/ml/matching_service.py:147
生成される理由:
• "スキルマッチ: React, TypeScript, Node.js"
• "希望勤務地: 東京都渋谷区"
• "希望給与条件を満たしています"
• "プロフィールとの総合的な適合度が高いです"
📊 STEP 8: ソート & Top-K選択
実装: app/ml/matching_service.py:299
処理:
1. match_score 降順でソート
2. 上位 K 件を選択 (デフォルト: 10件)
出力: JobRecommendation リスト
📤 STEP 9: レスポンス返却
形式: MatchingResponse (JSON)
内容:
• recommendations: [JobRecommendation]
• total_jobs: 元の求人数
• filtered_jobs: フィルタリング後の求人数

6. 実装コード例

6.1 マッチング実行（完全版）

# app/ml/matching_service.py:233 def recommend_jobs( self, seeker_profile: dict, available_jobs: List[dict], top_k: int = 10 ) -> List[JobRecommendation]: """求職者プロフィールに基づいて求人をレコメンド""" # STEP 1: 必須条件でフィルタリング filtered_jobs = self.filter_by_requirements(available_jobs, seeker_profile) if not filtered_jobs: return [] # STEP 2: 求職者プロフィールをベクトル化 seeker_text = self.embedding_service.create_seeker_text(seeker_profile) seeker_embedding = self.embedding_service.encode_text(seeker_text) # STEP 3: 各求人とのスコアを計算 recommendations = [] for job in filtered_jobs: # 求人をベクトル化 job_text = self.embedding_service.create_job_text(job) job_embedding = self.embedding_service.encode_text(job_text) # ベクトル類似度計算（0-80点） similarity = self.calculate_similarity(seeker_embedding, job_embedding) base_score = similarity * 80 # スキルマッチボーナス（0-20点） skill_bonus = self.calculate_skill_match_bonus(job, seeker_profile) # 最終スコア match_score = min(base_score + skill_bonus, 100.0) # マッチング理由を生成 match_reasons = self.generate_match_reasons(job, seeker_profile, match_score) recommendation = JobRecommendation( job_id=job.get("id", ""), job_data=job, match_score=match_score, match_reasons=match_reasons ) recommendations.append(recommendation) # スコア降順でソート recommendations.sort(key=lambda x: x.match_score, reverse=True) # Top-K を返す return recommendations[:top_k]

6.2 ベクトル埋め込み生成

# app/ml/embedding_service.py:28 def encode_text(self, text: str) -> np.ndarray: """テキストをベクトルに変換""" if not text or not text.strip(): return np.zeros(self.embedding_dim) embedding = self.model.encode(text, convert_to_numpy=True) return embedding # app/ml/embedding_service.py:103 def create_seeker_text(self, seeker_data: dict) -> str: """求職者プロフィールから埋め込み用のテキストを生成""" components = [] if seeker_data.get("skills"): skills = ", ".join(seeker_data["skills"]) components.append(f"スキル: {skills}") if seeker_data.get("experience"): components.append(f"経験: {seeker_data['experience']}") if seeker_data.get("location"): components.append(f"希望勤務地: {seeker_data['location']}") return " ".join(components)

6.3 コサイン類似度計算

# app/ml/matching_service.py:41 def calculate_similarity(self, vector1: np.ndarray, vector2: np.ndarray) -> float: """2つのベクトル間のコサイン類似度を計算""" v1 = vector1.reshape(1, -1) v2 = vector2.reshape(1, -1) similarity = cosine_similarity(v1, v2)[0][0] return float(similarity) # 0.0 ~ 1.0
